# MotherboardGuider

一个为 DIY PC 爱好者编写的轻量工具项目，旨在帮助装机时快速查找 **CPU 对应的主板型号** 和 **CPU+GPU 的大致功耗**，避免用户需要死记硬背这些固定但繁琐的知识点，提升装机配件搭配效率，尤其是选择电源瓦数时更精准。

---

## 🎯 功能简介
- 根据 CPU 系列快速匹配常见可用主板型号  
- 查看 CPU 与 GPU 搭配后的估算功耗  
- 动态展示支持的平台信息  
- 支持 Android 原生开发（Kotlin）  
- UI 简洁，适合快速查询  

---

## 📱 运行环境
- **最低支持版本：Android 7.0 (API 24)**
- 推荐使用 Android Studio 进行编译运行
- 开发语言：**Kotlin**
